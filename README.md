# 💪 Fitness App UI

A modern Fitness App UI built using **Flutter** and **Dart**, showcasing clean mobile application interfaces. The project focuses on creating an engaging user experience through onboarding screens, authentication pages, and fitness-themed user interfaces.

## 📖 Overview

This project demonstrates the design and implementation of a fitness mobile application interface using Flutter. It includes multiple screens commonly found in fitness and wellness applications, making it a great learning project for Flutter UI development and mobile app design.

<br>

## ✨ Features

### 🚀 Onboarding Experience

* Multi-screen onboarding flow
* Smooth page transitions
* Modern and user-friendly design
* Fitness-focused introduction screens

### 🔐 Authentication Interfaces

* Login Screen
* Registration Screen
* Clean form layouts
* Responsive user interface

### 🎨 Modern UI Design

* Custom layouts and components
* Attractive color scheme
* Mobile-friendly design
* Responsive Flutter widgets

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/tehzeeb-kousarr/fitness-app.git
```

### Navigate to Project Directory

```bash
cd fitness-app
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```
